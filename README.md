# Weather-Informator
## 📝 О проекте
>[!IMPORTANT]
> **Важно!** ☁️Проект, реализованный на основе библиотеки ***"python-telegram-bot"***, позволяет проинформировать пользователя о том, какая сейчас погода☁️.  
Реализовано это все с помощью:
- **Асинхронных функций**
- **Бесконечных циклов**
- **Command-Handler`ов**
- **Различных файловых импортов и т.д.**

Данный код включает в себя большое множество асинхронных функций, "фильтров" для времени, импортов различных библиотек, несколько регулярных выражений и т.п... **Как это работает:**

<ins>Пользователю предоставляется две возможности получать информацию о состоянии погоды, которые работают следующим образом:</ins>
> [!IMPORTANT]
> :memo: Получение информации о "состоянии неба" изначально дается из json на английском языке, затем, записывается в переменную status :memo:.
  1) :mag:Команда  **`/start`** позволяет узнать о погоде прямо сейчас. 
  2) :loud_sound:Команда **`/addtime`** вызывает асинхронную функцию, которая просит пользователя ввести время (можно ввести несколько и через запятую). Если же пользователь ввел команду повторно после добавления времени, то ему показывается, какое время у него уже есть, чтобы он не запутался (работает это через флаг MessageHandler`а, и, весьма успешно!).
  3)Затем, пользователь должен ввести команду /sub, чтобы запустить рассылку по тому времени, которое он добавил. Рассылка запускается, и все работает! (Просто запускается бесконечная функция которая отслеживает раз в минуту то время, которое добавил пользователь. Само время тут достаточно странное. Имеется в виду то, что отправка оповещения идет строго по определенной последовательности цифр. Не допустимо, если у нас в словарь время 08:00 будет сохранено. Необходимо его отфильтровать, чтобы потом, это же время: 8:0 поставить в цикл проверки и рассылки!). Если рассылка запущена не будет, то сообщение не придет!

  **Остальные команды:**
  - При необходимости удалить время, нужно ввести команду **`/removetime`**, которая покажет, какое время у вас есть, чтобы вы случайно не ввели "несуществующее" время.
  - Команда **`/showtime`** показывает время, в которое вам приходит рассылка.
  - Команда **`/uns`** позволяет отписаться от рассылки. Работает это достаточно просто: вызывается MessageHandler, который распределяет введенное время именно в асинхронную функцию для удаления вреимени(благодаря "флагу на удаление"). Затем, время просто удаляется через remove.
