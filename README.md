# Weather-Informator
Проект, реализованный на основе библиотеки python-telegram-bot, позволяет проинформировать пользователя о том, какая сейчас погода.  Реализовано это все с помощью асинхронных функций, бесконечных циклов, Command-Handler`ов, различных файловых импортов и т.д. И хоть код выглядит достаточно маленьким, всего на 100 с лишним строк, но включает в себя сразу несколько и другх библиотек, которые работают в паре друг с другом и обеспечивают пользователя необходимой информацией в нужное время!
Информация бралась исключительно из открытых источников, таких как StackOwerFlow, Habr, docs-python, сайты яндекс-практикума и т.д.
